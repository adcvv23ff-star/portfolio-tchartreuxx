# TCHARTREUXX Portfolio

Portfolio cá nhân của **Nguyễn Bảo Trung** (MSSV: 25041189) — Sinh viên khoa Ngôn ngữ và Văn hóa Anh, Trường Đại học Ngoại ngữ – Đại học Quốc gia Hà Nội.

Xây dựng cho học phần **Nhập môn Công nghệ số và AI**, tổng hợp 6 bài thực hành trong 6 tuần học.

## 🌐 Xem trực tiếp

Sau khi deploy (xem `HUONG-DAN-DEPLOY.md`), portfolio sẽ có link dạng:
`https://<tên-tài-khoản>.github.io/<tên-repo>/`

## 📁 Cấu trúc

```
├── index.html          # Trang chủ / Giới thiệu
├── du-an.html           # Trang Dự án (danh sách 6 bài)
├── bai-1.html … bai-6.html   # 6 trang bài thực hành chi tiết
├── tong-ket.html         # Trang Tổng kết
├── assets/
│   ├── profile.jpg       # Ảnh đại diện
│   ├── favicon-*.png     # Icon tab trình duyệt
│   ├── bai1/ … bai6/      # Ảnh minh họa từng bài
│   └── pdfs/              # File PDF sản phẩm từng bài
└── HUONG-DAN-DEPLOY.md   # Hướng dẫn đưa web lên mạng miễn phí
```

## ✨ Tính năng

- Thiết kế Gen Z / Bold, gradient tím–hồng–cam, font Space Grotesk + Inter
- Chế độ **Dark/Light mode** (nút bật/tắt trên navbar, tự nhớ lựa chọn)
- **Lightbox** xem ảnh minh họa phóng to (bấm vào ảnh trong các trang Bài)
- Hiệu ứng cuộn mờ dần (scroll-reveal), sakura rơi, mascot gốc, sticker trôi nổi
- Breadcrumb điều hướng, nút "về đầu trang", thanh tiến độ đọc
- Timeline 6 tuần học phần ở trang Dự án
- Liên kết mạng xã hội (Facebook, Instagram, Threads, Gmail) ở trang chủ
- Nút tải/xem PDF sản phẩm ngay trong từng trang bài

## 🛠️ Công nghệ

HTML thuần + Tailwind CSS (qua CDN) — không cần build, không cần server, mở trực tiếp `index.html` là chạy được.

## 📄 Giấy phép

Nội dung học thuật thuộc về tác giả. Vui lòng không sao chép nội dung bài làm cho mục đích gian lận học thuật.
